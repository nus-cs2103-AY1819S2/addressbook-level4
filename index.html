Address Book (Level 4)

Regular expense trackers too complicated to use? Introducing to you, ePiggy! A simplified expense tracker application catered to make saving money a breeze for young children and students!

Want to find out more? Go to the User Guide to find out more!

ePiggy is an open source project that welcomes contributions from the community. If you would like to contribute, go to the Developer Guide to get started!
