Welcome to HealthHub!

An administrative tool designed to ease the hassle of patient-doctor appointments.

To start registering a request, type
add request n/‹patient_name› i/‹identity_no› p/‹phone_no› a/‹address› dt/‹date_time› c/‹condition›.


To start registering a health care professional, type
add healthworker n/‹hw_name› i/‹identity_no› p/‹phone_no› o/‹organisation› s/‹specialisation›.


For more help navigating the application, type help.
